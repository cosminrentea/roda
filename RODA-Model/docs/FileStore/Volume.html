NAME

FileStore::Volume =cut

VERSION

version 0.01 =cut

DESCRIERE

Sistem de manipulare a directoarelor si fisierelor. Deocamdata stocate intr-un volum de tip filesystem Ideea este ca nici una dintre operatiile cu fisiere facute prin intermediul acestui modul (cu exceptia initializarii) sa nu opereze cu cai absolute. Toate operatiile cu directoare si fisiere, cu cateva mici exceptii se fac utilizand obiecte de tipul Path::Class

SYNOPSIS 

  use FileStore::Volume;
  use Path::Class;

  my $fs = FileStore::Volume->new(rootpath => '/my/root/path'); #nu folosim Path::Class aici, ci un simplu string care insa va fi automat convertit in Path::Class

  my $newdir = $fs->create_path(dir('ok','for','me'));
  
  my $isnfile = $fs->writefile(dir('ok/for/me'), $testfile1); 
  
  my $file = $fs->writepathfile(dir('my','new','directory'), $testfile1);
